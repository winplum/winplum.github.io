  4. 核对栏显示为“手动”时，下拉“记账科目”栏选科目

832下单→支付确认→网银汇款
  1. 采购下单
    2.

    在平台选择商品下单，填写并核对订单信息，确认无误后点击“生成订单”。

  2. 订单页面生成确认
    3.

    点击“生成订单”后弹出订单支付重要提示：支付时请从平台获取付款方式（点击支付后查看），勿向供应商其他账号付款。重要提示阅读完后点击“我已知晓”→跳转“采购人中心”。→“先款后货”订单需要在15天内支付完毕，到期未付款订单自动取消；→“先货后款”订单请在收到货及发票后，点击验货后支付货款。

  3. 支付环节~单笔支付
    4.

    操作流程：采购人中心→待支付货款→找到商品订单→点击“支付货款”→弹出支付页面→点击“支付货款”。

  4. 支付环节~合并支付
    5.
    1. 同一家店铺多笔订单支持合并支付，并生成一个公用识别码，付款时填上即可。不同店铺多笔订单暂不支持合并支付。
    2. 操作流程：采购人中心→待支付货款→找到需要支付的多笔商品订单→点击右上角“加入合并支付”→点击“支付货款”。
  5. 支付环节~点击支付
    6.
    1. 点击支付后，会弹出收款账号和识别码显示页面，财务人员根据页面内容到网银向收款账户进行汇款。
    2. 然后登录网银操作，汇款时务必将汇款识别码单独填写至所有用途/备注/摘要栏/附言等可填写栏目。
    3. 转账成功后无需在平台上传凭证，半小时后订单状态会自动变成已支付。
  6. 支付环节~注意事项
    7.
    1. 下单后请尽快转账，务必保证转账金额与订单金额一致，请勿多转、少转和分次转账，否则将因为汇款信息有误而导致支付失败，钱款将自动退回。
    2. 汇款时要正确填写汇款识别码至汇款单"用途/备注/摘要"等栏，认真核对收款人户名、银行卡号、开户银行等信息，汇款信息有误将导致支付失败，钱款将自动退回。
    3. 汇款识别码有效期共为10天，请尽快在10天有效期内完成支付，对于先款后货的订单，若因支付超时导致订单取消，则识别码也同步失效。
    4. 有效期过后，旧识别码失效，继续支付需重新申请新识别码，以最新申请的识别码为准进行汇款，否则视为识别码错误，订单支付失败，钱款将自动原路退回。
    5. 转账成功后无需在平台上传凭证，半小时后后订单状态会自动变更。
